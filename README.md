# Christmas Wishlist Application

This is a Streamlit application for managing a Christmas wish list. Users can create a personalized wish list, add items with gift names and purchase links, and indicate whether they want to gift the item. The application also includes password protection for added security.

## Features

- Create and name your wish list.
- Add items with a gift name and a purchase link.
- Checkbox to indicate "I want to gift this" which disables editing for that item.
- Secure your wish list with a password.

## Installation

1. Clone the repository:
   ```
   git clone <repository-url>
   cd christmas-wishlist
   ```

2. Install the required packages:
   ```
   pip install -r requirements.txt
   ```

## Usage

1. Run the application:
   ```
   streamlit run app.py
   ```

2. Enter the password to access your wish list.

3. Use the interface to add items, manage your wish list, and indicate gifting preferences.
